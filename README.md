# cashmemosystem-vityarthi
CASHMEMO SYSTEM
Overview
This project is a simple Python-based billing system that creates a cash memo for small shops. It collects customer details, item details,calculates totals and prints a formatted cash memo in the console. The system is built using two Python modules: one for customer details and one for the billing workflow.

Features
* Input shop name and invoice number
* Automatic customer ID generation
* Add multiple items with quantity and price
* Auto-calculation of item totals and grand total
* Clean and readable cash memo output
* Modular structure with separate customer module

Technologies / Tools Used
* Python 3
* Standard input/output
* No external libraries required

Steps to Install & Run
1. Download or clone the project folder.
2. Ensure both files exist in the same directory:
* Cashmemo system.py
* Customer.py
3. Open terminal or command prompt in the project directory.
4. Run the main program:
5. python "Cashmemo system.py"
6. Follow the on-screen prompts to enter shop, customer and item details.

Instructions for Testing
* Test with different customer names and phone numbers to check ID generation.
* Try various item counts to confirm that totals are calculated correctly.
* Enter different quantities and prices to verify the grand total.
* Check output formatting for readability.
